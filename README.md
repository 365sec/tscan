# tscan
NB Scanner for ipv4 space 

````
go version go1.8.3 linux/amd64

安装步骤：

1、 go  get github.com/chushuai/tscan

将会安装tscan 到   $GOPATH/src/github.com/chushuai/tscan   目录下

cd $GOPATH/src/github.com/chushuai/tscan/cmd

go build

```
